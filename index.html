My Projects

Find My Climate

Find My Climate is a project that aims to pair up users with their preferred climate conditions. A user can adjust a number of sliders that filter various geographical and meteorlogical conditions such as record temperature, elevation, or precipitation. This project is currently being refactored.

Cryptocurrency Trader

With crypto markets open 24/7 I thought it would be a fun challenge to see if I could create a program that could generate a profit consistantly. I use a risk minimized version of a martingale betting strategy to lower my average buy in price. This project is functional but not currently in use.

Quizzing GUI

My brother is a bird watcher and was looking for a way to memorize a large number of images quickly. Though Quizlet exists I thought it would be a good learning experience to make a more customizeable version and to learn how to make a GUI in python. This project is functional and needs minor adjustments.
